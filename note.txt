npm install yin 3 myo install lo ya
-----------------------------------

1. global (ko yae computer tae mar htae lite tae ta baw , server por tin yin ma par twar buu)
2. dev (ko yae project tae win twar tar, server por tin yin ma par twar buu)
3. dependencies (project tae mar shit mar, server por tin yin ma par buu)

*****dependencies ka har bae server por mar par twar mar
